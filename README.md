# roblox-skills

Skills for AI coding agents to help with Roblox game development.

## Install

```bash
npx skills add sentinelcore/roblox-skills
```

## Skills

### `roblox-monetization`

End-to-end guide for earning Robux through legitimate Roblox monetization systems. Covers both Roblox Studio scripting and Creator Hub dashboard setup.

**Includes:**
- Game Passes (one-time purchases, server-side ownership checks)
- Developer Products (repeatable purchases, DataStore persistence, idempotency)
- UGC / Avatar Items (marketplace setup, earning rates, moderation)
- Premium Payouts (passive income from Premium subscribers)
- DevEx (converting Robux to real money, eligibility requirements)

```bash
# Install just this skill
npx skills add https://github.com/sentinelcore/roblox-skills/tree/main/roblox-monetization
```
